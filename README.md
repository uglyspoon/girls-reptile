## 应用介绍
PicGather是基于electron-vue开发的个人图片收藏器，支持自定义收藏源站配置，兼容MacOS与Windows操作系统
## 下载安装
您可以[点击此处](https://github.com/Licoy/pic-gather/releases)进行下载安装，支持Mac/Windows/Linux(未测试)
